Python script to create a Telegram bot.

The goal of this script is to obtain lab info and simplify life.

## COMMAND SYNTAX
/[COMMAND]

Available commands:  
  `lab`  : Show the number of people in lab.  
  `log` : Show the complete log (only for admin user).
